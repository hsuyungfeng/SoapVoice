# 醫療語音測試腳本 - 8個場景

## 錄音須知
- 使用普通話（台灣國語）
- 語速適中，每句稍微停頓
- 每個檔案約 20-30 秒
- 環境安靜，距離麥克風約 15-20 公分

---

## 1. 胸痛 (chest_pain.wav)

```
醫師：請坐，今天有什麼不舒服？
患者：醫師，我昨天開始胸口很悶，呼吸不太順暢。
醫師：這樣的情況持續多久了？有沒有發燒或咳嗽？
患者：大概兩天了，沒有發燒，但是走路快一點就會喘。
醫師：之前有類似症狀嗎？有没有心臟方面的病史？
患者：沒有，但是我有抽菸，大概抽了十年。
醫師：好，我先幫你安排心電圖和胸部X光檢查。
患者：醫師，嚴重嗎？
醫師：目前先做檢查確認，你不要緊張。
```

## 2. 高血壓 (hypertension.wav)

```
醫師：張先生，今天來血壓控制怎麼樣？
患者：醫師，最近血壓還是很高，早上都160多。
醫師：有按時吃藥嗎？鹽分有沒有控制？
患者：藥有吃，但是口味比較重，鹽放比較多。
醫師：這樣不行喔，高血壓要好好控制，不然會影響腎臟和心臟。
患者：醫師，我有時候會頭暈，是不是血壓太高？
醫師：對，血壓高的時候容易頭暈，我幫你調整一下用藥。
患者：好的，那我要注意什麼？
醫師：少鹽、少油、多運動，每天量血壓記錄下來。
```

## 3. 糖尿病 (diabetes.wav)

```
醫師：李女士，你的血糖控制怎麼樣？
患者：醫師，這幾天血糖都比較高，空腹大概200多。
醫師：飲食有沒有按時？水果吃多少？
患者：有吃飯，但是忍不住會吃水果，上午還吃了兩顆芭樂。
醫師：芭樂雖然不甜，但是糖分還是有，要適量。
患者：醫師，那我可以吃什麼水果？
醫師：芭樂、蓮霧、奇異果比較好，一次不要超過一碗。
患者：醫師，我的腳有時候會麻，是不是神經病變？
醫師：有可能，這是糖尿病常見的併發症，我幫你安排神經檢查。
```

## 4. 傷口護理 (wound_care.wav)

```
護理師：阿姨，我來幫你換藥。
患者：護理師，這傷口什麼時候會好？
護理師：我看一下傷口，恢復得不錯，已經沒有發炎了。
患者：已經一個禮拜了，本來說三天就可以洗澡對吧？
護理師：對，手術傷口十天後就可以碰水了，這兩天先擦澡。
患者：傷口旁邊有點癢，正常嗎？
護理師：傷口在癒合會癢，盡量不要抓，我幫你塗止癢藥膏。
患者：請問傷口上的美容膠帶要撕掉嗎？
護理師：讓它自然脫落就好，不用刻意撕，平時傷口保持乾燥。
```

## 5. 呼吸道 (respiratory.wav)

```
醫師：怎麼了？咳嗽嗎？
患者：醫師，我咳嗽一個多禮拜了，有痰但是咳不出來。
醫師：有没有發燒？喉咙痛不痛？
患者：有一點發燒，喉咙很痛，痰是黃色的。
醫師：先去照一下胸部X光，看一下肺部情況。
患者：醫師，我是不是肺炎？
醫師：目前不確定，等X光結果出來再說，我先幫你抽血檢查。
患者：好，那需要住院嗎？
醫師：先看檢查結果，如果肺部有發炎可能需要住院治療。
```

## 6. 醫囑 (drug_order.wav)

```
醫師：王先生，這是你的處方籤。
患者：醫師，這些藥怎麼吃？
醫師：這包是降血壓藥，每天早餐後吃一顆。這包是利尿劑，下午吃，注意要多喝水。
患者：醫師，這個止痛藥什麼時候吃？
醫師：痛的時候再吃，一天最多四顆，不要空腹吃，會傷胃。
患者：這些藥有沒有副作用？
醫師：可能會有點頭暈或是手腳無力，如果有問題立刻回診。
患者：好的，這些藥要吃多久？
醫師：血壓藥先吃一個月再回診，利尿劑看情況可以停。
```

## 7. 手術記錄 (surgery_record.wav)

```
醫師：手術開始時間上午九點二十分。
護理師：確認病人姓名，陳大明，手術項目是腹腔鏡膽囊切除術。
醫師：確認麻醉完成，開始消毒手術部位。
醫師：建立第一個穿刺孔於肚臍上方，插入腹腔鏡。
醫師：確認膽囊位置，發現有輕微發炎黏連。
護理師：出血量約50cc，手術進行中。
醫師：分離膽囊與周圍組織，準備夾住膽管。
醫師：確認沒有其他異常，開始切除膽囊。
醫師：取出膽囊，手術傷口縫合，手術時間四十五分鐘。
醫師：病人生命徵象穩定，手術完成。
```

## 8. 醫病對話 (doctor_patient.wav)

```
患者：醫師，我最近很容易疲倦，晚上睡不好。
醫師：這樣多久了？壓力大嗎？
患者：大概一個月了，工作很忙，然後一直加班。
醫師：會不會心悸、手抖？
患者：有一點，有時候脾氣很不好。
醫師：我幫你驗血，看一下甲狀腺功能。
患者：醫師，我是不是甲狀腺有問題？
醫師：先檢查再說，甲狀腺亢進會造成這些症狀。
患者：如果真的是甲狀腺問題要怎麼治療？
醫師：先吃藥控制，严重的话可能需要開刀或是做放射碘治療。
```

## 錄音設備設定建議

| 項目 | 建議值 |
|------|--------|
| 取樣率 | 16kHz 或 44.1kHz |
| 格式 | WAV (PCM) |
| 音量 | -12dB ~ -6dB |
| 背景 | 安靜環境，避免迴音 |
| 距離 | 15-20 cm |
