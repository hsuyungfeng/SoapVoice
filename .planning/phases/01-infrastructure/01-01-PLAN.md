<task type="auto">
  <name>建立硬體驗證腳本</name>
  <files>scripts/verify-hardware.sh</files>
  <action>
建立 scripts/verify-hardware.sh 腳本，實現以下驗證：

1. GPU 檢測
   - 執行 nvidia-smi 檢測 NVIDIA GPU
   - 解析 VRAM 大小 (需 ≥44GB)
   - 解析 GPU 數量

2. RAM 檢測
   - 讀取 /proc/meminfo 或使用 free -b
   - 驗證總記憶體 ≥48GB

3. 儲存檢測
   - 檢查根分區或指定路徑的可用空間
   - 驗證可用空間 ≥500GB

4. 輸出格式
   - JSON 格式輸出驗證結果
   - 每項檢測包含: name, passed, value, required, message

範例輸出:
{
  "gpu": {"passed": true, "count": 2, "total_vram_gb": 88, "required": 44},
  "ram": {"passed": true, "total_gb": 48, "required": 48},
  "storage": {"passed": true, "available_gb": 500, "required": 500}
}
  </action>
  <verify>bash scripts/verify-hardware.sh</verify>
  <done>腳本執行成功，輸出符合預期的 JSON 格式，所有檢測項目通過</done>
