# SoapVoice 專案核心價值

## 項目概述

建立一套基於本地 LLM 的醫療語音轉 SOAP 病歷系統，實現即時多語語音輸入、AI 驅動的醫療語意標準化、結構化 SOAP 病歷自動產出，以及 ICD-10 診斷代碼映射。

## 核心價值主張

- **醫師效率提升**: 減少 60-80% 病歷撰寫時間 (8-10min → 3-4min)
- **品質提升**: 標準化醫療術語，減少錯誤和歧義
- **資料結構化**: 利於後續醫療數據分析與統計
- **成本控制**: 本地部署，無雲端 API 費用，完全開源技術棧
- **患者隱私**: 所有資料保留在內網，不涉及外部傳輸
