## 🎯 快速分類技巧

### 識別每個類別的特徵詞

```
主訴 (S) 特徵詞：
└─ 患者說...、主訴...、抱怨...、症狀...、不舒服...

客觀 (O) 特徵詞：
└─ 檢查、測量、檢驗、結果、數值、發現、顯示、顯著...

評估 (A) 特徵詞：
└─ 診斷、確診、疑似、可能、應該是...、初步判斷...

計劃 (P) 特徵詞：
└─ 建議、處方、開立、開始、繼續、停止、避免...
```

## 📝 實戰演練

### 例句分類練習

**例1**: "患者頭痛3天，體溫39度。"
```
S: "患者頭痛3天" ← 患者症狀(關鍵詞: 頭痛)
O: "體溫39度"    ← 檢查結果(關鍵詞: 體溫)
```

**例2**: "驗血結果顯示白血球12000，診斷為感冒，開立退燒藥。"
```
O: "驗血結果顯示白血球12000" ← 檢驗結果(關鍵詞: 白血球)
A: "診斷為感冒"               ← 診斷(關鍵詞: 診斷、感冒)
P: "開立退燒藥"               ← 治療計畫(關鍵詞: 開立、退燒藥)
```

**例3**: "血壓偏高，初步認為高血壓。建議低鹽飲食和定期監測。"
```
O: "血壓偏高"                 ← 客觀檢查(關鍵詞: 血壓)
A: "初步認為高血壓"           ← 診斷(關鍵詞: 初步、高血壓)
P: "建議低鹽飲食和定期監測"   ← 治療計畫(關鍵詞: 建議、低鹽、監測)
```

## 🔍 常見混淆區別

### S vs O: 症狀 vs 檢查結果
```
❌ 錯誤混淆: "患者說血壓高" → 這是主訴
✅ 正確分類:
   - S: "患者說感覺到頭暈"    ← 患者主觀感受
   - O: "血壓測量值 160/100"  ← 客觀測量數據
```

### A vs P: 診斷 vs 計畫
```
❌ 錯誤混淆: "診斷為肺炎，需要治療"
✅ 正確分類:
   - A: "診斷為社區型肺炎"  ← 診斷結論
   - P: "開立抗生素治療"    ← 治療計畫
```

### O vs P: 檢查結果 vs 治療計畫
```
❌ 錯誤混淆: "建議每週驗血一次"
✅ 正確分類:
   - O: "上週驗血結果HbA1c 7.2" ← 已經做過的檢查結果
   - P: "建議每週驗血一次"      ← 未來要做的計劃
```

## 💡 關鍵詞記憶方法

### 按詞性分類

**名詞(症狀)** → 更可能是 **S**
```
頭痛、疲勞、咳嗽、發燒 = 患者症狀
```

**動詞(醫學行為)** → 考慮 **O**
```
檢查、測量、拍攝、注射 = 已執行的檢查
```

**形容詞(診斷)** → 更可能是 **A**
```
高血壓、糖尿病、肺炎、感染 = 診斷病名
```

**祈使動詞(建議)** → 更可能是 **P**
```
開立、建議、避免、定期、繼續 = 治療計畫
```

## 📱 使用場景

### 場景1: 門診初診
```
主訴: "頭痛3天、發燒、喉嚨痛" (患者說)
客觀: "體溫39℃、血壓升高" (醫生量)
評估: "初步診斷為流感" (醫生判斷)
計劃: "開立退燒藥、建議居家休息" (醫生建議)
```

### 場景2: 慢性病追蹤
```
主訴: "血糖控制還可以、無新症狀" (患者報告)
客觀: "空腹血糖120、HbA1c 7.2" (檢驗結果)
評估: "糖尿病控制穩定" (醫生判斷)
計劃: "繼續原藥、定期回診、飲食控制" (醫生建議)
```

### 場景3: 急診評估
```
主訴: "胸痛1小時、呼吸困難" (患者症狀)
客觀: "血壓160/100、心跳110、EKG異常" (檢查結果)
評估: "急性冠心症疑診" (醫生判斷)
計劃: "急診住院、肌肉注射嗎啡、準備進開刀房" (醫生計劃)
```

## ✅ 驗證清單

在使用分類器時，檢查：

- [ ] **S** 包含患者自述的症狀和感受嗎？
- [ ] **O** 包含具體的測量數值和檢驗結果嗎？
- [ ] **A** 包含明確的診斷名詞和臨床判斷嗎？
- [ ] **P** 包含治療建議、用藥和後續計畫嗎？
