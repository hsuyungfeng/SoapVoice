- 短音訊（<1分鐘）差異不明顯

## 4. 建議配置

## 5. 已知問題

### RTX 2080 Ti 雙卡相容性

- **qwen3.5:9b** 在雙卡設置下會崩潰
- 其他模型正常運作
- 解決：使用 qwen2.5 系列或 phi4

---

## 6. 結論

**首選推薦：** `qwen2.5:14b`
- 9GB 大小適中
- 中文醫療術語表現最佳
- 處理時間 8.4s 可接受

**替代方案：**
- 硬體受限 → `qwen2.5:3b` (3.4s)
- 需要快速反饋 → `qwen2.5:7b` (8.6s)
